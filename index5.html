Complete CSS Tutorial

CSS (Cascading Style Sheets) is used to style HTML elements, control layouts, colors, fonts, animations, and more.

1. CSS Syntax

CSS rules consist of a selector and a declaration block:

Copied!
h1 {
  color: blue;
  font-size: 24px;
}

2. Selectors

Target elements using type, class, ID, and attribute selectors:

Copied!
p { color: gray; }
#header { background-color: #f4f4f4; }
.container { padding: 20px; }
a[href^="https"] { color: green; }

3. Colors & Backgrounds

Set colors using names, HEX, RGB, or HSL:

Copied!
body { background-color: #f8f9fa; }
h1 { color: #007bff; }
div { background-image: url('image.jpg'); }

4. Fonts & Text

Control font family, size, weight, style, alignment, and decoration:

Copied!
p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: bold;
  text-align: center;
  text-decoration: underline;
}

5. Box Model

All elements are boxes: content, padding, border, margin:

Copied!
div {
  margin: 10px;
  padding: 15px;
  border: 1px solid #000;
}

6. Positioning

Control element positions:

Copied!
div {
  position: relative;
  top: 10px;
  left: 20px;
}

.fixed-box {
  position: fixed;
  bottom: 0;
  right: 0;
}

7. Flexbox

Flexbox is used for one-dimensional layouts:

Copied!
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

8. CSS Grid

Grid is used for two-dimensional layouts:

Copied!
.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 10px;
}

9. Transitions & Animations

Animate property changes or create keyframe animations:

Copied!
button {
  transition: background-color 0.3s;
}

button:hover {
  background-color: #155ab6;
}

@keyframes slide {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}

.box {
  animation: slide 2s infinite alternate;
}

10. Pseudo-classes & Pseudo-elements

Style elements based on state or part:

Copied!
a:hover { color: red; }
li:first-child { font-weight: bold; }
p::first-line { text-transform: uppercase; }

11. Media Queries

Make your layout responsive for different screen sizes:

Copied!
@media (max-width: 768px) {
  body { font-size: 14px; }
  .container { flex-direction: column; }
}
