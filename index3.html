HTML Tutorial

HTML (HyperText Markup Language) is the standard language for creating web pages. It consists of elements that define content and structure.

1. HTML Document Structure

Every HTML document starts with a <!DOCTYPE html> declaration and has a <html>, <head>, and <body> section:

Copied!
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

2. Headings

HTML has six levels of headings:

Copied!
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

3. Paragraphs and Text Formatting

Use <p> for paragraphs. Inline formatting includes <strong>, <em>, <mark>, etc.

Copied!
<p>This is a <strong>strong</strong> and <em>emphasized</em> paragraph.</p>

4. Links

Copied!
<a href="https://www.example.com">Visit Example</a>

5. Images

Copied!
<img src="image.jpg" alt="Description" />

6. Lists

Copied!
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

7. Forms

Copied!
<form action="/submit" method="post">
  <label>Name:</label>
  <input type="text" name="name" />
  <input type="submit" value="Submit" />
</form>

8. Tables

Copied!
<table border="1">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Alice</td><td>25</td></tr>
</table>

9. Semantic Elements

HTML5 semantic elements:

  • <header>, <footer>
  • <article>, <section>, <aside>
  • <nav>

10. Multimedia

Copied!
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls width="320">
  <source src="video.mp4" type="video/mp4">
</video>

11. Best Practices

  • Use semantic elements.
  • Always close tags.
  • Add alt for images.
  • Keep code indented.
  • Use lowercase for tags/attributes.
